## 이 코스에 대해

- **목적**: 미국 상장 기업의 공시 정보를 경쟁사 분석에 활용하기 위한 **도메인 지식** 전수
- **대상**: 전략 투자 분석가, 실사 팀, M&A 연구자, 상장 경쟁사를 모니터링하는 모든 사람
- **형식**: 9개의 독립 HTML 파일, 각 레슨 약 10~15분, 레슨당 3개의 퀴즈
- **사례 기업**: Carvana (CVNA), Carmax (KMX), AutoNation (AN), Uxin (UXIN), AutoHome (ATHM)
- **라이선스**: CC BY 4.0 — 공유, 변경, 저작자 표시
