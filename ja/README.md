# SEC Lessons — 日本語版

> **🌐 ようこそ / Welcome / Bienvenue / Willkommen / Bienvenido / 환영합니다 / 欢迎**

これは、米国上場企業をSEC提出書類（10-K、10-Q、8-K、Form 4、13F、13D/G）を通じて監視し、競合分析に活用するための **9レッスンシリーズ** の日本語版です。

## コースの使い方

1. **ここから始める**: [`lessons/0001-sec-filing-101.html`](./lessons/0001-sec-filing-101.html) — 10-K、10-Q、8-Kの基礎
2. **推奨順序**: 0001 → 0002 → 0003 → 0004 → 0005 → 0006 → 0007 → 0008 → 0009
3. **チートシートを手元に**: [`reference/sec-form-types.html`](./reference/sec-form-types.html) は印刷可能
4. **クイズを飛ばさない** — よくある判断ミスを浮き彫りにするために設計されています

## このコースについて

- **目的**: 米国上場企業の開示情報を競合分析に活用するための**領域知識**を教える
- **対象者**: 戦略投資アナリスト、デューデリジェンスチーム、M&Aリサーチャー、上場企業の競合を監視するすべての人
- **形式**: 9つの独立したHTMLファイル、各レッスン約10〜15分、各レッスンに3つのクイズ
- **事例**: Carvana (CVNA)、Carmax (KMX)、AutoNation (AN)、Uxin (UXIN)、AutoHome (ATHM)
