中古車業界の核心財務指標
10-K/10-Q から何を、なぜ見るのか

10-K には数百の財務指標が含まれているが、競合モニタリングにすべてを見る必要はない。中古車業界において本当に重要なのは十数個だけだ。肝心なのは、各指標が中古車業界で何を意味するかを理解すること——これはどの財務教科書にも書いていない。

第一優先順位：収益構造

Carvana 2026Q1: Retail 収益 $5.86B、Wholesale 収益 $470M、その他 $92M

中古車会社の収益は「車を1台売って1回お金を受け取る」ほど単純ではない。重要な問いは、どこからお金を稼いでいるのか？である。

収益タイプ意味戦略的意義
小売収益 (Retail Revenue)C 端：個人消費者への販売中核事業。成長率と ASP（平均販売価格）を注視
卸売収益 (Wholesale Revenue)B 端：オークション/ディーラーへの卸売売れ残り車の卸売——卸売比率上昇は小売在庫問題の兆候
金融収益 (Finance/Other Revenue)ローン手数料、保険、延長保証など利益率の隠れた源泉。急成長は必ずしも良い兆候ではない（融資品質低下の可能性）
収益は総額だけでなく、構造を見るべき。卸売収益の比率が突然上昇したら、小売が売れなくなっている証拠。

第二優先順位：収益性——中古車業界は2つで十分

売上総利益率（Gross Margin）は聖杯

中古車は低利益率のビジネス。SaaS 企業の 70% 超の売上総利益率とは異なり、中古車会社の売上総利益率は通常 15〜20%。なぜ売上総利益率を最も注視すべきか？

  • 売上総利益率は1台あたりの売買差益を反映（GPU = Gross Profit Per Unit、1台あたり粗利）
  • 売上総利益率の変動は純利益よりも問題を明確に示す——純利益は一時的な費用で歪む可能性がある
  • 売上総利益率の継続的な低下 = 仕入原価の上昇 + 販売価格の据え置き不能 + 在庫滞留による値下げの悪循環
実データ：Carvana 2026Q1 Gross Profit = $1.27B、Revenue = $6.43B
→ Gross Margin = 1.27 ÷ 6.43 = 19.8%
比較：従来型ディーラー Carmax の売上総利益率は通常 12〜14%（Carmax は「価格交渉なしの一律価格」が基本で、利益幅は狭いが回転率は高い）

SG&A 対収益比率

SG&A（販売費・一般管理費）は中古車会社の最大の費用項目。なぜ SG&A/収益比率が純利益率よりも情報量が多いのか？

  • SG&A には広告宣伝費、店舗運営費、人件費が含まれる
  • SG&A/収益比率の低下 = 規模の経済が機能し、1台あたりの固定費が逓減
  • SG&A/収益比率の上昇 + 収益成長 = 企業が成長のために積極的に資金を投下——「この成長は持続可能か」と問う必要がある
実データ：Carvana 2026Q1 SG&A = $690M、Revenue = $6.43B
→ SG&A/Revenue = 10.7%
この比率は 2024 年には約 12% であり、Carvana の運営効率が改善していることを示している。

第三優先順位：貸借対照表——中古車会社が最も恐れるもの

中古車会社の命運は損益計算書ではなく、貸借対照表にある。

指標なぜ重要か危険信号
現金 (Cash)中古車は資金集約型ビジネス。車の仕入に多額の現金が必要現金 / 四半期収益 < 0.5 → 流動性懸念
長期負債 (Long Term Debt)中古車会社は在庫仕入を借入に依存負債の伸びが収益の伸びを上回る → 借り換えに依存
在庫 (Inventory)在庫＝車——長期保管すると価値が下がる（iPhone とは違う！）在庫日数の急増 → 車が売れず、値引き販売が不可避に
中古車業界の鉄則：車は在庫に1日置かれるごとに少しずつ減価する。Carmax が低利益率でも高回転を追求する理由はここにある——在庫回転率は1台あたりの利益よりも重要である。
実データ：Carvana 2026Q1 Cash = $2.41B、Total Assets = $13.77B、Long Term Debt = $4.92B（FY2025データ）
現金/四半期収益 = 2.41 ÷ 6.43 = 0.37 → この比率は危険ではない（車両仕入の支払いタイミングと販売タイミングは完全には一致しない）が、継続的な注視が必要。

Quiz

Q1: 中古車業界において、売上総利益率（Gross Margin）が純利益よりも重要な理由は？

クリックして答えを表示

売上総利益率は1台あたりの売買差益（中核事業能力）を反映し、一時的な費用、資本構成、税制などの要因に影響されない。純利益は債務再編益などで「一見良好」に見えても中核事業が悪化している可能性があるが、売上総利益率はごまかせない。中古車のような低利益率業界では、売上総利益率の1〜2%の変動が会議を開く価値がある。

Q2: ある中古車会社の卸売収益（Wholesale）比率が突然5%から15%に上昇した場合、何を示唆するか？

クリックして答えを表示

小売側に問題が発生している可能性——車が売れず、卸売で在庫処理せざるを得ない。卸売価格は小売価格を大幅に下回るため、卸売比率の上昇は次のことを示す：① 在庫管理に問題（売れにくい車を仕入れすぎた）；② 市場需要の低迷；③ 値引き在庫処分の可能性があり、結果的に売上総利益率のさらなる低下を招く。一言で言えば、卸売収益比率の上昇は良いニュースではない。

Q3: なぜ中古車会社にとって「在庫回転率」は一般の小売企業よりも重要なのか？

クリックして答えを表示

中古車は減価するからだ。在庫に60日置かれた車と30日置かれた車では価値が異なる——コカコーラが倉庫で劣化しないのとはわけが違う。さらに中古車価格は市場の需給に非常に敏感で、在庫滞留＋市場冷え込み＝大幅値下げ＝売上総利益率の急落となる。Carmax のビジネスモデル（一律価格＋迅速な在庫回転）は、この構造的な矛盾を解決するためにある。
