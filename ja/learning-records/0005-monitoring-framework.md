## Key insights

1. **競合モニタリングの四層構造**: 財務諸表 (what) → イベント駆動 (when what) → 経営陣のシグナル (why) → クロスカンパニー比較 (so what)。システムが前三層を担当し、人間が第四層を担当する。
2. **5社の選定ロジック** — 各社のポジショニングは異なる：
   - Carvana: 「純粋オンラインモデル」の成否を検証
   - Carmax: オフライン最適モデルの効率ベンチマーク
   - AutoNation: 産業チェーン上流（新車→中古車）の観測窓口
   - Uxin: 中国市場の特殊な問題のサンプル
   - AutoHome: 業界トラフィックのバロメーター
3. **単独企業のトレンド分析よりもクロスカンパニーでの相互検証の方が価値が高い** — シグナルマトリックス：A 社の X 指標の変化 + B 社の Y 指標の同時変化 → 結論 Z を導き出せる
4. **再帰性の視点**: 自社の filing も競合にモニタリングされている — 各 filing を作成する際には「競合がこれを読んでどう行動するか」を考えるべき
5. **システムの究極的価値**: 生データを構造化されたインテリジェンスに変換する → 人間がそのインテリジェンスを自社の戦略判断に活かす
