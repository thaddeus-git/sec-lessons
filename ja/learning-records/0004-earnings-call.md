# Learning Record 0004 — Earnings Call の戦略的価値

**Date**: 2026-06-16
**Lesson**: 0004-earnings-call.html
**Status**: 完了

## Key insights

1. **Prepared Remarks はリハーサルされたパフォーマンスであり、Q&A こそ台本のない本音の会話である**
2. **Guidance の文言変化は数字の変化よりも先行する** — 「自信がある」→「慎重」への変化は、数字の下方修正より 1-2 四半期先行することが多い
3. **ガイダンスの撤回 (guidance withdrawal) は悪い数字よりも深刻** — 経営陣自身も将来が見通せないことを意味する（visibility = 0）
4. **経営陣が何に答えるかよりも、アナリストが何を質問するかが市場の懸念を示している**
5. **「全体的なトーン」が最も過小評価されている判断材料** — 数字に問題がなくても「どこかおかしい」と感じたら → 直感を信じる

## Reading order for EC summary (5 min)

1. 業績ハイライト (30s) → 2. ガイダンスの変更 (1min) → 3. Q&A の重要な質疑応答 (2min) → 4. リスクの提示 (1min) → 5. 全体の判断 (30s)
