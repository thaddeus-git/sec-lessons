# Learning Record 0002 — Indicateurs financiers clés du secteur des véhicules d'occasion

**Date** : 2026-06-16
**Leçon** : 0002-financial-metrics-used-car.html
**Statut** : Terminé

## Points clés

1. **La structure des revenus est plus importante que le revenu total** — L'évolution de la répartition entre le détail, le gros et les services financiers révèle souvent un changement dans la santé de l'activité. Une part croissante du gros = des problèmes dans le segment du détail.
2. **La marge brute est le saint graal du secteur des véhicules d'occasion** — Elle reflète la véritable rentabilité par véhicule, sans être perturbée par les charges exceptionnelles. Plage normale dans le secteur : 15-20 %.
3. **Le ratio SG&A/Revenu est l'indicateur clé de l'efficacité opérationnelle** — Une baisse indique que les économies d'échelle se concrétisent ; une hausse indique que l'entreprise brûle de l'argent pour croître.
4. **La rotation des stocks est plus importante pour les concessionnaires de véhicules d'occasion que pour le commerce de détail en général** — Les voitures se déprécient ; un stock excédentaire = baisse des prix forcée = effondrement de la marge brute.
5. **Signaux d'alarme dans le bilan** — Trésorerie/Revenu trimestriel < 0,3, croissance de la dette > croissance du revenu → risque de liquidité.
